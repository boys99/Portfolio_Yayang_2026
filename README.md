## Cara Menjalankan Secara Lokal

Proyek ini tidak memerlukan proses instalasi apapun. Cukup buka file `index.html` menggunakan browser modern seperti Chrome, Firefox, atau Edge.

```bash
git clone https://github.com/boys99/portfolio.git
cd portfolio
# Buka index.html di browser
```

Jika ingin menjalankan melalui local server:

```bash
# Python 3
python -m http.server 8000
# Akses melalui http://localhost:8000
```
